Point-wise F1
각 시점을 독립적으로 분류. 가장 보수적이며 해석이 명확하나 구간 이상 위치 오차에 가혹합니다.
Point-Adjusted F1
정답 구간 내 하나라도 탐지 시 전체 인정. 관대하여 자명한 예측도 높은 점수를 받을 수 있습니다.
합성 평가 한계
탐지기가 노린 유형으로 평가하면 성능이 낙관적으로 나올 수 있습니다. 실제 라벨과의 갭에 주목하세요.
알고리즘 · 탐지기 6종
합의 투표 메커니즘
하드 투표: 각 탐지기가 이상(1)/정상(0)을 판정하고 ≥N표면 이상 확정
임계 자동 선택: 합성 주입 라벨 기준 F1 최대 지점으로 탐지기별 임계 자동 추정
N 선택: N을 1~6으로 바꿔가며 F1을 계산, 균형점을 사이드바에서 조절 가능
votes_i = Σk [ score_k,i ≥ θk ]
anomaly_i = 1 iff votes_i ≥ N*
N* = argmax_N F1(votes ≥ N)
알려진 한계
변수 간 상관 이상 미탐지
각 변수를 독립 단변량 시계열로 분석합니다. 변수 간 상관 구조가 깨지는 이상은 감지하지 못합니다. PCA 재구성 오차 기반 탐지기를 추가하면 보완됩니다.
합성 평가의 낙관 편향
탐지기가 노린 유형(spike·shift·variance·drift)으로 평가하므로 실제 성능보다 높게 나오는 경향이 있습니다.
오프라인 배치 전제
전체 시계열을 한 번에 받아 분석하는 배치 구조입니다. 실시간 스트리밍 온라인 탐지는 지원하지 않습니다.
