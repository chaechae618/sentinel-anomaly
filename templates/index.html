02 · 탐지기 합의 분석

Point-wise F1
각 시점을 독립적으로 분류. 가장 보수적이며 해석이 명확하나 구간 이상 위치 오차에 가혹합니다.
Point-Adjusted F1
정답 구간 내 하나라도 탐지 시 전체 인정. 관대하여 자명한 예측도 높은 점수를 받을 수 있습니다.

탐지기가 노린 유형으로 평가하면 성능이 낙관적으로 나올 수 있습니다. 실제 라벨과의 갭에 주목하세요.
알고리즘 · 탐지기 6종
합의 투표 메커니즘
하드 투표: 각 탐지기가 이상(1)/정상(0)을 판정하고 ≥N표면 이상 확정
임계 자동 선택: 합성 주입 라벨 기준 F1 최대 지점으로 탐지기별 임계 자동 추정
N 선택: 사이드바 슬라이더로 1~6표 조절 가능
votes_i = Σk [ score_k,i ≥ θk ]
anomaly_i = 1 iff votes_i ≥ N*
N* = argmax_N F1(votes ≥ N)
